⚙️ Configuration Examples

Default Configuration (No setup needed):

<script src="https://your-domain.com/market-engine-widget.js"></script> <div id="market-engine-widget"></div>

Just add these two lines and you're done!

With Dark Theme:

<script> window.MARKET_ENGINE_CONFIG = { THEME: 'dark' }; </script> <script src="https://your-domain.com/market-engine-widget.js"></script> <div id="market-engine-widget"></div>

Update Every 1 Minute Instead of 30 Seconds:

<script> window.MARKET_ENGINE_CONFIG = { AUTO_UPDATE_INTERVAL: 60000 // 60 seconds }; </script> <script src="https://your-domain.com/market-engine-widget.js"></script> <div id="market-engine-widget"></div>

Run in Background (No Visual Display):

<script src="https://your-domain.com/market-engine-widget.js"></script> <script> // Access data from JavaScript document.addEventListener('marketEngineUpdate', (e) => { console.log('Market data:', e.detail); }); </script>

Widget runs in background without displaying anything. Access data via JavaScript.

📱 JavaScript API

Access and control the widget from JavaScript:

Get Market Data

const data = window.MarketEngine.getData(); console.log(data.companies); console.log(data.timestamp);

Get Company

const company = window.MarketEngine .getCompany('TechCorp Inc'); console.log(company.roi_percentage);

Listen to Updates

window.MarketEngine.on('update', (data) => { console.log('Updated:', data.timestamp); });

Control Updates

const instance = window.MarketEngine.instance; instance.stopAutoUpdate(); instance.startAutoUpdate();

💡 Integration Examples

Example 1: Track Total Portfolio Value

window.MarketEngine.on('update', (data) => { const totalValue = data.companies.reduce( (sum, company) => sum + company.current_capital, 0 ); document.getElementById('total').innerText = '$' + totalValue.toLocaleString(); });

Example 2: Alert on Profit

window.MarketEngine.on('update', (data) => { data.companies.forEach(company => { if (company.total_profit_loss > 5000) { console.log(`🎉 ${company.name} is up!`); } }); });

Example 3: Send to Analytics

window.MarketEngine.on('update', (data) => { if (window.gtag) { gtag('event', 'market_update', { companies: data.total_companies, timestamp: data.timestamp }); } });
